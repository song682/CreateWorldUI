# CreateWorldUI

A modern world creation screen for **Minecraft 1.7.10**, backported from newer versions of the game.  
This mod redesigns the entire world creation UI to match the clean, intuitive experience of modern Minecraft.

If you feel the vanilla 1.7.10 world creation menu is outdated or cluttered, CreateWorldUI gives it a fresh, modern look.

---

## ✨ Features

### **Modern Tabbed Layout**
World creation options are reorganized into three tabs:
- **Game** – game mode, difficulty, cheats  
- **World** – seed, world type, structure generation  
- **More** – additional settings and future extensions  

This layout mirrors modern Minecraft and makes navigation much easier.

---

### **Built‑in GameRule Editor**
Edit gamerules directly in the UI:
- Toggle buttons for boolean rules  
- Text fields for numeric or string rules  
- Reset and Save buttons for quick adjustments  

No more typing `/gamerule` commands manually.

---

### **UI Improvements**
- Cleaner layout for world name, seed, and buttons  
- Custom textures for tabs and gradients  
- **Backported modern difficulty selection**, including difficulty lock  
- Overall smoother and more intuitive user experience

---
